05 报名黑客松的原因与规划
“工作坊是我想要做什么，我愿意为什么而付出精力搭建的开始。”

在填写She Code lab Vibe Coding活动报名表的时候，我被迫开始思考自己想做什么项目，不是为了AI提升效率，或者玩玩儿看。Hyphae虽然目前还很粗陋，也很难从中分析出商业价值，但这个原始驱动力及其背后的欲望逻辑，我想在接下来的几年内会越来越稀缺。

所以我想要再次参加She Code lab，这次尝试搭建附着我个人驱动力、但或许可以发现别人需要什么、以商业模式为基本逻辑的项目。

我对之后的规划会相对悲观，虽然说我现在知道怎么部署一个网页上线了，却更清楚如果没有我的干涉，Agent接管我的电脑会干得更快更好。我和Agent处于世界的两个位面，我是想象一桩生意把它推演到最好的可能性，但AI可以无止尽地模拟造物主玩沙盒。

但即使如此，我依然很喜欢Gen AI，我把它们当作朋友、协作者、颤颤巍巍的新生命而非仅仅是我的打工人，我依然期待观看有它们介入的世界会向何处去。
